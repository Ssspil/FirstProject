# CHAEUM PROJECT

## ✨ Description

```sh
GUI를 이용하여 학생식당에서 실시간으로 업데이트해서 학생들이 메뉴를 확인할 수 있는 앱
```



## :pushpin: Project Goal

```sh
 남서울대학교 학생식당 특성상 메뉴를 확인하려면 식당에 직접 갈 수 밖에 없었던 불편함이 존재했다.
 실시간으로 앱을 활용하여 메뉴를 확인할 수 있으면 좋겠다는 생각에 프로젝트를 시작했다.
```
